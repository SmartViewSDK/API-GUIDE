com.samsung.multiscreen

Class Service

  • java.lang.Object
    • com.samsung.multiscreen.Service

  • public class Service
extends Object
    Encapsulates a MultiScreen-enabled service. Also provides the entry point for communication with services. May be created when services are discovered via Search, or by getByURI(Uri, Result) and getById(Context, String, Result), if the service url or id are known and specified, respectively.
