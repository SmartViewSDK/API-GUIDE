com.samsung.multiscreen

Class Player

  • java.lang.Object
    • com.samsung.multiscreen.Player
  • Direct Known Subclasses:
    AudioPlayer, PhotoPlayer, VideoPlayer

    public class Player
extends Object
    The Application class allows control Default Media Player (DMP) on TV side. It provides basic APIs to control DMP and acts as base class for deriving classes - Audio, Video and Photo player classes.
