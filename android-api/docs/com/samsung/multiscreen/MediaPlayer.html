com.samsung.multiscreen

Class MediaPlayer

  • java.lang.Object
    • com.samsung.multiscreen.MediaPlayer

  • public class MediaPlayer
extends Object
    The MediaPlayer class allows control default Media Player on TV side It provides API to send launch, play content commands, pause, stop, mute, unmute, forward, rewind, set volume, seek to specified position. It receives events regarding Media Player states from TV side.
